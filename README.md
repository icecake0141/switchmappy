# switchmappy

Python 3.12+ reimplementation of the Perl-based `switchmap` tooling.

## Installation

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .
```

Optional features:

```bash
pip install -e .[snmp,search]
```

Development dependencies:

```bash
pip install -e .[dev]
```
